杂记

本机安装的是Python3.7

  1. Python解释器
  2. pip包管理器，管理的是扩展的一些库文件（默认存在Python安装目录\Lib\site-packages下）
  3. Python的标准库：Lib

pip包管理器：

  • 安装一个库文件：pip install 库名

    默认是去官方国外网站下载，速度慢

    可以采用国内镜像源：

    • 清华：https://pypi.tuna.tsinghua.edu.cn/simple

    • 阿里云：http://mirrors.aliyun.com/pypi/simple/

    • 中国科技大学 https://pypi.mirrors.ustc.edu.cn/simple/

    • 华中理工大学：http://pypi.hustunique.com/

    • 山东理工大学：http://pypi.sdutlinux.org/

    • 豆瓣：http://pypi.douban.com/simple/

    临时使用：

    • pip install 库名称 -i 镜像源地址

    永久使用:

    • windows环境：直接在user目录中创建一个pip目录，再新建文件pip.ini，例如：C:\Users\LJ\pip\pip.ini

    • Linux环境：修改 ~/.pip/pip.conf (没有就创建一个文件夹及文件。文件夹要加“.”，表示是隐藏文件夹)，pip.ini文件内容同上面windows

  • 卸载一个库文件：pip uninstall 库名
